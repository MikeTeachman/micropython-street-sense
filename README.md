# Street Sense

Street Sense is a MicroPython based project to build a portable, battery-powered sensor unit to measure: 

- **Air Quality**
    - Ozone
    - NO2 
    - Particulates 
- **Noise Pollution**

## Dependencies
### Hardware 
Lolin D32 Pro  
<https://wiki.wemos.cc/products:d32:d32_pro>
### Micropython
